Class: ListNode

ListNode(data)

A single node of a doubly linked list. Contains data and a reference to the next node.

Constructor

new ListNode(data)

Creates a new node containing the given data. The next node is set to undefined and gets updated by the DoublyLinkedList class.
Parameters:
Name Type Description
data number The data stored by this node.
